5 ストライプ模様の背景

横のストライプ

  • まず、#fb3 から #58a への縦方向の線形グラデーションを用意する（下左図）。
      background: linear-gradient(#fb3, #58a);
  • linear-gradient() では、以下のようにカラーストップの位置を指定可能。（下中央図）
      background: linear-gradient(#fb3 20%, #58a 80%);
  • カラーストップをまったく同じ位置に指定すると、グラデーションではなくなる。（下右図）
      background: linear-gradient(#fb3 50%, #58a 50%);
  • このグラデーションは背景画像と同等に扱えるため、background-size でサイズ変更が可能。背景画像は繰り返し表示されるため、コンテナ全体にストライプが表示される。（下左図）
      background: linear-gradient(#fb3 50%, #58a 50%);
      background-size: 100% 2rem;
  • 同じカラーストップの値を繰り返し記述する代わりに、2つ目の色のカラーストップをゼロに指定すると1つ目と一致するように位置が調整される。（下中央図）
      background: linear-gradient(#fb3 50%, #58a 0);
      background-size: 100% 2rem;
  • カラーストップの位置を50%から変えると、幅の異なるストライプを生成可能。（下右図）
      background: linear-gradient(#fb3 30%, #58a 0);
      background-size: 100% 2rem;
  • 3色以上のストライプも、次のように簡単に作成可能。
      background: linear-gradient(#fb3 33.3%, #58a 0, #58a 66.6%, yellowgreen 0);
      background-size: 100% 3rem;

縦のストライプ

  • 縦ストライプのためのコードは横のストライプと同じだが、グラデーションの方向を表すパラメータ（"to right" というキーワード、または、90deg）を追加している。
  • さらに、キーワード指定では端から間隔を空けられなかった。
      background: linear-gradient(to right, #fb3 50%, #58a 0);
      background-size: 2rem 100%;
      background: linear-gradient(90deg, #fb3 50%, #58a 0);
      background-size: 2rem 100%;

斜めのストライプ

  • "90deg" の指定で縦のストライプを生成できたので、45度の斜めのストライプは "45deg" を指定すればよいと思われるかもしれない。しかし、これは機能しない。（下左図）
  • これは、グラデーションの単位となるタイルを個別に回転しているだけで、背景全体が回転しているわけではないから。
      background: linear-gradient(45deg, #fb3 50%, #58a 0);
      background-size: 2rem 2rem;
  • １つのタイルの中に、4つのストライプを指定すれば、斜めのストライプにはなる。しかし、横や縦のストライプよりも線幅が狭い。（下右図）
      background: linear-gradient(45deg, #fb3 25%, #58a 0, #58a 50%, #fb3 0, #fb3 75%, #58a 0);
      background-size: 2rem 2rem;
  • background-size はタイルの斜辺の長さに対応するので、ピタゴラスの定理より目的のストライプの幅の2√2の値を指定すればよいということになる（下左図）
      background: linear-gradient(45deg, #fb3 25%, #58a 0, #58a 50%, #fb3 0, #fb3 75%, #58a 0);
      background-size: 2.828427rem 2.828427rem;
  • 有限小数は正確に表現不可能なので、小数点以下は丸めてシンプルことを推奨。（下右図）
      background: linear-gradient(45deg, #fb3 25%, #58a 0, #58a 50%, #fb3 0, #fb3 75%, #58a 0);
      background-size: 2.8rem 2.8rem;
  • 上記のストライプ指定方法は、柔軟ではない。たとえば45度ではなく60度など他の角度を指定すると、ひどい結果になってしまう。（下図）
      background: linear-gradient(60deg, #fb3 25%, #58a 0, #58a 50%, #fb3 0, #fb3 75%, #58a 0);
      background-size: 2.8rem 2.8rem;
  • linear-gradient() には繰り返し表示するための repeating-linear-gradient() がある。
  • これを使うと以下のように記述する代わりに、 
          background: linear-gradient(45deg, #fb3, #58a 20px, #58a 0, #fb3 40px, #fb3 0, #58a 60px, ...);
    以下のように記述できる。（下左図） 
          background: repeating-linear-gradient(45deg, #fb3, #58a 20px);
  • 45度のストライプを繰り返しのグラデーションを使うと次のように記述できる。（下左図）
      background: repeating-linear-gradient(45deg, #fb3, #fb3 10px, #58a 0, #58a 20px);
  • さらにこの記述では、任意の角度を指定可能。（下中央図）
      background: repeating-linear-gradient(60deg, #fb3, #fb3 10px, #58a 0, #58a 20px);
  • ストライプの線幅も考慮すると、45度の斜めのストライプは以下のように記述可能。（下右図）
      background: repeating-linear-gradient(45deg, #fb3 0, #fb3 25%, #58a 0, #58a 50%);
      background-size: 2.8rem 2.8rem;

明るさを変えたストライプ

  • 異なる色のストライプではなく、同じ色で明るさだけが異なる次のようなストライプを考える。（下左図）
      background-image: repeating-linear-gradient(30deg, #79b, #79b 10px, #58a 0, #58a 20px);
  • 暗い方の色を背景色にし、半透明の白のストライプを重ねることで、可読性と柔軟性を向上できる。（下右図）
      background: #58a;
      background-image: repeating-linear-gradient(30deg, hsla(0,0%,100%,.1), hsla(0,0%,100%,.1) 10px, transparent 0, transparent 20px);
