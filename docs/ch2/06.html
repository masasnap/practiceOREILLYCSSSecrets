6 複雑な背景のパターン

  • CSS3 Patterns Gallery では、2011年時点のグラデーションで実現可能な機能を紹介している。
  • 以下では、比較的作りやすく使いやすいものをピックアップする。

格子模様

  • 縦（下左図）と横（下中央図）のストライプを組み合わせることで、格子模様が作れる（下右図）
      background: white;
      background-image: linear-gradient(90deg, rgba(200,0,0,.5) 50%, transparent 0),
      		linear-gradient(rgba(200,0,0,.5) 50%, transparent 0);
      background-size: 30px 30px;
  • 線の幅を一定に保ちたい場合は、カラーストップの位置として%値ではなく長さを指定する（下左図）。
      background: #58a;
      background-image: linear-gradient(white 1px, transparent 0),
      		linear-gradient(90deg, white 1px, transparent 0);
      background-size: 30px 30px;
  • 更に幅や色の異なる線を組み合わせると、よりリアルな格子を作成できる（下右図）。
      background: #58a;
      background-image: linear-gradient(white 2px, transparent 0),
      		linear-gradient(90deg, white 2px, transparent 0),
      		linear-gradient(hsla(0,0%,100%,.3) 1px, transparent 0),
      		linear-gradient(90deg, hsla(0,0%,100%,.3) 1px, transparent 0);
      background-size: 75px 75px, 75px 75px, 15px 15px, 15px 15px;

水玉模様

  • 円形グラデーションを使うと、ドットが表現できる（下左図）。
      background: #655;
      background-image: radial-gradient(tan 30%, transparent 0);
      background-size: 30px 30px;
  • 同じグラデーションを2つ組み合わせると、綺麗な水玉模様が表現できる（下右図）。
      background: #655;
      background-image: radial-gradient(tan 30%, transparent 0),
      	radial-gradient(tan 30%, transparent 0);
      background-size: 30px 30px;
      background-position: 0 0, 15px 15px;

市松模様

  • 1つのCSSグラデーションでは、余白のある四角形は作成できない。
  • そこで、直角三角形を組み合わせて四角形を表示する。
  • 直角三角形は以下のように作成できる（下左図）。
      background: #eee;
      background-image: linear-gradient(45deg, #bbb 50%, transparent 0);
      background-size: 30px 30px;
  • カラーストップの位置を50%から25%にすると、下中央図のようになる。カラーストップの位置を反転させると、下右図のようになる。
  • 2つのグラデーションを組み合わせると、下左図のようになる。
  • 2つ目のグラデーションの位置をずらし、それぞれの三角形を組み合わせて四角形にする。（下中央図）
      background: #eee;
      background-image: linear-gradient(45deg, #bbb 25%, transparent 0),
      	linear-gradient(45deg, transparent 75%, #bbb 0);
      background-position: 0 0, 15px 15px;
      background-size: 30px 30px;
  • このグラデーションをもう1組用意すると、下右図のようになる。
      background: #eee;
      background-image: linear-gradient(45deg, #bbb 25%, transparent 0),
      	linear-gradient(45deg, transparent 75%, #bbb 0),
      	linear-gradient(45deg, #bbb 25%, transparent 0),
      	linear-gradient(45deg, transparent 75%, #bbb 0);
      background-position: 0 0, 15px 15px, 15px 15px, 30px 30px;
      background-size: 30px 30px;
  • 反対向きの2つの三角形（1&4個目、2&3個目）をそれぞれ1つのlinear-gradientにまとめると、下図のようになる。
      background: #eee;
      background-image: linear-gradient(45deg,
      		rgba(0,0,0,.25) 25%, transparent 0,
      		transparent 75%, rgba(0,0,0,.25) 0),
      	linear-gradient(45deg,
      		rgba(0,0,0,.25) 25%, transparent 0,
      		transparent 75%, rgba(0,0,0,.25) 0);
      background-position: 0 0, 15px 15px;
      background-size: 30px 30px;
  • 上記は複雑なので、実際にはSVGの利用を薦める（下図）。
      background: #eee url('data:image/svg+xml,\
      		<svg xmlns="http://www.w3.org/2000/svg" width="100" height="100" fill-opacity=".25">\
      		<rect x="50" width="50" height="50" /> \
      		<rect y="50" width="50" height="50" /> \
      		</svg>');
      background-size: 30px 30px;
