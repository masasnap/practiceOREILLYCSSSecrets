4 角の内側を丸める

  • ボーダーの外側は直角のままで、内側だけを丸めたいケース。
  • 要素を２つ追加すれば表示可能だが、本質的には要素は１つあれば十分なはず。
      .ex4-1 {
        background: #655;
        padding: .8rem;
      }
      .ex4-1 > div {
        background: tan;
        border-radius: .8rem;
        padding: 1rem;
        max-width: 9rem;
      }
I have a nice subtle inner rounding, don't I look pretty?
  • 「ボーダー」が単色でよい場合には、１つの要素だけで済ませられる（ハックがある）。
  • outline で要素の丸めを無視し、box-shadow で隙間を埋める。
        outline: .6rem solid #655;
        box-shadow: 0 0 0 .4rem #655;
        border-radius: .8rem;
        padding: 1rem;
        background: tan;
I have a nice subtle inner rounding, don't I look pretty?
  • 補足
    • outline だけを表示すると、下左図のように（現状のレンダリングエンジンでは）角の丸めが反映されない。
    • box-shadow でオフセットもぼかしもない影を設定すると、下中央図のように角の丸めが反映される。
    • 両者を合わせると、下右図のように目的の表示が得られる。
