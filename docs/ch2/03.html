3 柔軟な背景の位置指定

  • Background & Borders Level 3 (w3.org/TR/css3-background) では、複数のボーダーを指定不可。
  • border-image を使わずに CSS のみで表現できないか？

[1] box-shadow を使う

  • box-shadow は、カンマ区切りで値を複数指定可能
      background: yellowgreen;
      box-shadow: 0 0 0 10px #655,
                  0 0 0 15px deeppink,
                  0 2px 5px 15px rgba(0,0,0,.6);
  • inset キーワード指定で、要素の内側にも影を描画できる
  • border-radious にも追従する
  • 制約: 影はレイアウトに影響を与えない、イベントが発生しない、実線しか表現できない

[2] outline を使う

  • ボーダーが2本だけ必要なのであれば、アウトラインで表現可能
      background: yellowgreen;
      box-shadow: 0 0 0 10px #655,
                  0 0 0 15px deeppink,
                  0 2px 5px 15px rgba(0,0,0,.6);
  • 制約: 3本以上のボーダーが表現不可、レイアウトに影響を与えない、イベントが発生しない、border-radius が考慮されない
