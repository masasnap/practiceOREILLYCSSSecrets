3 柔軟な背景の位置指定

  • CSS2.1では背景画像のオフセットは左上基点かキーワード指定しかできない。
  • さらに、キーワード指定では端から間隔を空けられなかった。
      background-position: right bottom;
Code Pirate

[1] background-position 拡張を使う

  • Background & Borders Level 3 (w3.org/TR/css3-background) では、背景画像に対しすべての隅からのオフセットを指定可能。
      background-position: right bottom;
      background-position: right 20px bottom 10px;
Code Pirate

[2] background-origin を使う

  • Background & Borders Level 3 (w3.org/TR/css3-background) の background-origin プロパティを利用すると、どのボックス (border-box, padding-box, cotnent box) の隅から配置するかを指定可能。
      background-position: right bottom;
      background-origin: content-box;

[3] calc() を使う
